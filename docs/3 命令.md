### 3、node 命令

- `node`

  - 功能：
    1. 打开 node 的 console 控制台
    2. 执行某个 js 文件

- `nodemon`

  - 功能：解决频繁修改代码，所带来的频繁重启服务器的问题。每次修改代码之后，nodemon 能够帮助我们自动重启服务器。

  - 注意：该命令属于第三方命令行工具，需要独立安装

  - 安装：

  - ```shell
    npm install --global nodemon
    ```

  - 使用：

  - ```shell
    nodemon app.js
    ```
